
# An addictively chaotic bullet-hell shooter where you mow down wave after wave of geometric foes. Dive into mayhem: blast nine enemy-circle types with seven weapons and six skills, face The Snake, earn a loyal pet, and unlock fairy buffs to turn the tide!

Visit [Itch.io page](https://xlorenz.itch.io/zero-pi)


## Features
- 9 enemy types: from simple normies to the huge Juggernaut  

- 7 weapons: burst shot, shotgun, double barrel, and more  

- 6 active skills: shield, dash, implosion...  

- Climactic boss fight against The Snake  

- Collect a cute pet reward on victory  

- Fairy buffs spawning mid-battle  

- Full character & scenery customization in the store  

- Save your run and return whenever


( game made entirely without graphic assets, in python, using pygame )
